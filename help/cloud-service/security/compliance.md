# 法規遵循與認證

[!DNL Adobe Commerce as a Cloud Service]是建立在信任、透明以及嚴格遵守全球安全性和隱私權標準的基礎之上。 Adobe的企業級安全性計畫可確保[!DNL Adobe Commerce as a Cloud Service]符合受監管的產業和注重安全性的組織的期望。

## 產業認證

Adobe為支援[!DNL Adobe Commerce as a Cloud Service]的基礎結構和服務維護一套完整的認證和證明。 這些功能包括：

- **SOC 2 Type II**：獨立稽核會驗證Adobe對安全性、可用性和機密性的控制。

- **ISO/IEC 27001**： Adobe的資訊安全管理系統(ISMS)已在此全球公認的標準下通過認證。

- **PCI DSS**： [!DNL Adobe Commerce]基礎架構符合PCI DSS，確保付款資料的安全處理。 客戶有責任維護其自訂程式碼和擴充功能的PCI相容性。

- **GDPR和CCPA**： Adobe提供工具和檔案，協助客戶履行資料隱私權法規所規定的義務。

如需Adobe的認證與法規遵循檔案的完整清單，請造訪[Adobe信任中心](https://www.adobe.com/trust/compliance/compliance-list.html){target="_blank"}。

## 適用於從業者的認證

Adobe也為使用[!DNL Adobe Commerce]的開發人員、架構師和商務從業人員提供角色型認證。 這些認證可驗證在安全實作和平台控管方面的專業知識。 如需詳細資訊，請參閱[Adobe認證入口網站](https://experienceleague.adobe.com/en/docs/certification/program/technical-certifications/ac/ac-overview){target="_blank"}。
