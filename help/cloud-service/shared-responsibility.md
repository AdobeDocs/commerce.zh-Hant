# 共擔責任安全性與營運模式

{{accs-early-access}}

[!DNL Adobe Commerce as a Cloud Service]是隨選服務，需仰賴共擔責任安全性與運作模型。 這些責任由Adobe和客戶共用。 各方對保護及營運Adobe Commerce應用程式均有不同的責任。

>[!BEGINSHADEBOX]

下列摘要表使用RACI模型來顯示Adobe和客戶之間共用的安全性責任。

**R** — 負責
**A** — 負責
**C** — 已諮詢
**I** — 已通知

>[!ENDSHADEBOX]
