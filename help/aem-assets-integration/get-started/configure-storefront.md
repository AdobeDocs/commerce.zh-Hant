# 設定您的店面

AEM Assets整合會顯示在AEM Assets中管理的產品影像，而非傳統Commerce託管的影像。 此整合提供增強的影像管理功能：進階最佳化、裁切，以及透過Adobe的Content Delivery Network (CDN)傳遞。

若要搭配Edge Delivery Services支援的Commerce店面使用AEM Assets，請完成[EDS AEM Assets設定](https://experienceleague.adobe.com/developer/commerce/storefront/setup/configuration/aem-assets-configuration/)主題中所述的店面設定。
