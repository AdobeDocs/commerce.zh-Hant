# 同義字

同義字會擴充搜尋查詢，以納入購物者可能使用的與目錄中不同的字詞。 您不想因為某人正在尋找「沙發」，而您的產品卻被列為「沙發」而失去銷售機會。 您可以輸入客戶可能會用來尋找您的產品的所有可能字詞，以擷取廣泛的搜尋字詞。
